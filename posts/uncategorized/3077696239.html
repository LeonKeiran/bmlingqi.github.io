# 前言

大家是否有时候需要在一个文件夹内容寻找一个包含了特定内容的文件或者包含某段内容文件名的文件呢？用 Windows 文件浏览器自带的搜索呢？又太鸡肋了，百度呢？又找不到拥有该功能的软件。

于是，小编就开发了一款软件集成了这两种功能并给它命名 "FileFilter"，名称也通俗易懂，"File" 英译为: “文件”，“Filter"英译为" 过滤器”，因此该软件又名文件过滤器。

该软件采用不怎么火的 C# 开发，因此若遇见无法打开请检查您的电脑是否下载并安装了.NET 环境。
